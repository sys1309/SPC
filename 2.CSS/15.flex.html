메인 컨텐츠

여기는 메인 컨텐츠를 작성한느 곳입니다. 블라블라블라

  • 디스플레이(display): flex 외 디렉션(direction): column을 통해서 레이아웃을 수직으로 쌓을 수 있습니다.
  • main 안의 컨텐츠는 display:flex를 사용해서 좌우로 배치하고자합니다.
  • flex:1과 flex:3을 통해서 전체 페이지를 4등분해서 그중 하나를 네브바 영역으로, 나머지 3칸 영역을 아티클 영역으로 분배합니다.
  • 작은 화면(저해상도)에서는 flexbox의 flex-direction을 column으로 해서 네브바와 아티클을 서로 세로로 배치합니다.
